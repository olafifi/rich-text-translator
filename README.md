# Richly 富文本转换器

一个像翻译器一样工作的双向富文本转换网页：左侧可视化编辑，右侧 HTML 源码，支持双向转换和一键复制。

## 功能

- 可视化编辑：标题、字号、粗体、斜体、下划线、删除线
- 文字颜色、背景高亮、对齐、列表、行距
- 可视化富文本与 HTML 双向转换
- 一键复制富文本，直接粘贴到支持富文本的编辑器
- 一键复制 HTML 源码
- 深色 / 浅色主题与移动端适配
- 纯静态实现，内容仅在浏览器本地处理

## 本地使用

直接双击 `index.html`，或在目录中启动任意静态文件服务器。

例如：

```bash
python -m http.server 8080
```

然后访问 <http://localhost:8080>。

## 部署

项目不需要构建，可直接部署到 GitHub Pages、Vercel、Netlify 等静态托管服务。
