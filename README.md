# Richly 富文本转换器

一个像翻译器一样工作的双向游戏富文本转换网页：左侧可视化编辑，右侧 BBCode 风格标签，支持双向转换和一键复制。

在线使用：<https://olafifi.github.io/rich-text-translator/>

## 功能

- 可视化编辑：标题、字号、粗体、斜体、下划线、删除线
- 文字颜色、对齐、列表、行距
- 内置 HEX 取色器，固定显示大写 `#RRGGBB`，不依赖系统取色器的默认格式
- 可视化富文本与游戏标签双向转换
- 颜色输出格式：`[color=#FF4D4D]文字[/color]`
- 支持 `[b]`、`[i]`、`[u]`、`[s]`、`[size]`、`[align]` 等常用标签
- 一键复制游戏富文本代码或纯文本
- 深色 / 浅色主题与移动端适配
- 纯静态实现，内容仅在浏览器本地处理

## 本地使用

直接双击 `index.html`，或在目录中启动任意静态文件服务器。

例如：

```bash
python -m http.server 8080
```

然后访问 <http://localhost:8080>。

## 部署

项目不需要构建，可直接部署到 GitHub Pages、Vercel、Netlify 等静态托管服务。
